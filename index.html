How to Install

  1. Download the Extension:

    Click the button below to download the Botstrr extension ZIP file.

    Download Botstrr
  2. Unzip the File:

    Extract the contents of the ZIP file to a folder on your computer.

  3. Open Chrome Extensions Page:

    In Chrome, navigate to chrome://extensions/ or go to Menu > More Tools > Extensions.

  4. Enable Developer Mode:

    Toggle on the "Developer mode" switch in the top right corner of the Extensions page.

  5. Load Unpacked Extension:

    Click on "Load unpacked" and select the folder where you extracted the Botstrr files.

  6. Verify Installation:

    Ensure that the Botstrr extension appears in your list of installed extensions.
